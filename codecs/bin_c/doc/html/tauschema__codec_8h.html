Write to the location of iterator any finite value. When the iter already contains an element, it does verify it the tags match and the lengths match. When value is NULL, and length is 0 a null item will be written. When value is NULL and length is >0 all value field is replaced with 0x00

  • value - pointer to the value to copy the value from
  • len - length of the finite value memory field
